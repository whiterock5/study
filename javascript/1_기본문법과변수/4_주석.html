* 주석 : 코드문법에 영향없이 설명을 다는 것 1. html : 2. css : /* */ 3. javascript 1) 한줄주석 : // - 간단한 코멘트 처리 2) 여러줄주석 : /* */ - 코멘트가 길거나 코드를 잠시 실행시키고 싶지 않을 때 주석 처리 * 기본용어 1. 표현식(식) : 자바스크립트에서 값을 만들어내는 간단한 코드 2. ;(세미콜론) : 표현식의 종결의 의미 자바스크립트는 ;을 작성하지 않아도 실행문제가 없음 3. 키워드 : 자바스크립트가 처음 만들어질 때 정해진 특별한 의미가 있는 단어 4. 식별자 : 자바스크립트에서 개발자가 이름을 붙일 때 사용하는 단어 ex) 변수명, 함수명 등
