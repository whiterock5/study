* 객체 : 자바스크립트가 명령할 수 있는 모든 대상 1. 기본내장객체 : 기본적인 컴퓨터관련 객체 1) 문자열(String) : 문자열 2) 숫자(Number) : 숫자 3) 배열(Array) : 배열 - 값을 나열해서 담고 있는 것 4) 날짜시간(Date) : 서버가 갖고 있는 날짜와 시간을 갖고 있는 객체 5) 수학(Math) : 수식을 갖고 있는 객체 2. 브라우저 객체(BOM) : 브라우저 관련 객체 1) window - 브라우저 창 관련 객체 2) screen - 모니터 화면 객체 3) location - 주소표시줄 4) history - 이전다음버튼 5) navigator - 브라우저 정보(크롬, 익스플로러, 모바일접속 등등) 6) document - 문서객체(body내부 요소, 컨텐츠) 3. 문서객체(DOM) - 문서객체 모델 - body관련 객체 1) 요소 - 태그 2) 텍스트컨텐츠 3) 이미지컨텐츠 4) 동영상컨텐츠 5) 폼요소 4. 사용자 정의 객체 - 개발자가 만드는 객체
