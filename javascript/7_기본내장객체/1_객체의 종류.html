* 객체 : 자바스크립트가 명령가능한 모든 대상 1. 기본내장객체 : 자바스크립트가 기본 내장하고 있는 객체, 첫글자를 대문자 1) String : 문자열 객체 2) Number : 숫자 객체 3) Array : 배열 객체 4) Math : 수학 객체 5) Date : 날짜와 시간 관련 객체 2. BOM(브라우저관련) : 브라우저 관련 객체, 다 소문자로 작성 1) window : 브라우저 창관련 객체 2) location : 브라우저의 주소표시줄 객체 3) history : 브라우저의 이전다음 버튼 객체 4) screen : 모니터 화면 관련 객체 5) navigator : 브라우저 자체 정보 객체 - 크롬, 사파리, 모바일접속 등 6) document : body내부의 문서객체 3. DOM(문서객체관련) : body내부의 문서객체 1) 태그요소 2) 텍스트컨텐츠 3) 기타 - body내부에 들어가는 모든 객체 4. 사용자정의 객체 - 개발자가 직접 제작하는 객
