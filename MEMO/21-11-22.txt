--------

포워드는 상태표시줄에 안표시됨
같은 리퀘스트 객체를 사용하기 때문에

아작스 - > XHR이라는 객체 안에 리퀘스트객체를 넣음

넘어가면 같은 xhr 객체를 사용한다.

xhr 가 리퀘스트 가로채서 넘김.  포워드 아님/리다이렉트 아님.

넘어간 리퀘스트객체는 같은게아님.


모바일에서 DB 직접적 연동 불가능.

하지만 하려고하면 '파싱'이라는 것 이용해야함
파싱 가능 = xml , json

아작스 -> 가로채서 서블릿으로 보내줌
서로다른 환경의 언어들을 이어주는 레스트서버 대표적인게 아작스
실시간 = 요구할때마다  :: 요청하는 실제시간 (( ex 서버지정시간 3~5시 ))  (( 지정된 시간이 아닌것 ))
== 요청할때마다 접근가능한것.

아작스를 이용하면 DTO 를 건너지않고 직접 접근가능하다.

로그인할때 아작스쓰는게 편함.
아니면 댓글 모바일 아닐때.
편해서 쓴다. 무조건 아작스로 구현할필요없음.

list 자바 메모리에서 만듦
json 웹자체에서 만듦.

xhr 안의 리퀘스트는 리다이렉트방식 (내부적으로는)
새로운 리퀘스트객체를 생성한다.
